    • Breadcrumbs

    If you need them, Feeling Responsive offers you a breadcrumb navigation. You can easily turn it on/off via frontmatter.

    WARNING: If you use breadcrumbs be aware, that if you use categories, you need pages for those categories. If not, the link in the breadcrumb won't work.

    Front Matter Code

    breadcrumb: true
